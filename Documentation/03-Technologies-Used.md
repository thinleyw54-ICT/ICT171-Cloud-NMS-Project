# Technologies Used

## Microsoft Azure
Cloud platform used to host the Ubuntu Linux Virtual Machine.

## Ubuntu Linux
Operating system used for server administration and website hosting.

## Nginx
Web server used to host the project website.

## Netdata
Real-time monitoring platform used to monitor CPU, memory, disk, and network performance.

## GitHub
Used for version control, project documentation, and source code management.

## DNS
Used to connect the custom domain to the Azure server.

## HTTPS (Certbot SSL)
Provides secure encrypted communication between users and the website.
