# Website Development

## Overview

The project website was developed using HTML and CSS and hosted on an Ubuntu Linux Virtual Machine using the Nginx web server.

The website provides project information, assignment documentation, monitoring dashboard access, and student details through a professional web-based interface.

---

## index.html

The `index.html` file serves as the homepage of the Cloud-Based Network Monitoring System website.

Features:

* Project title and introduction
* Cloud-Based Network Monitoring System overview
* Navigation sidebar
* Project objectives
* Live monitoring dashboard link
* Professional user interface

The homepage acts as the central entry point for users accessing the website.

---

## assignment1.html

The `assignment1.html` page contains the Assignment 1 Project Proposal.

Features:

* Project proposal documentation
* Cloud monitoring solution overview
* License rationale
* Monitoring dashboard access
* Academic project information

This page demonstrates the planning and design phase of the project.

---

## assignment2.html

The `assignment2.html` page provides access to the Assignment 2 presentation.

Features:

* Presentation information
* Canva presentation link
* Project demonstration materials
* Supporting documentation

This page showcases the presentation component of the project.

---

## about.html

The `about.html` page contains student information and project background.

Features:

* Student details
* Project information
* Academic information
* Personal introduction

This page identifies the project author and provides background information.

---

## style.css

The `style.css` file provides the visual design and layout for the website.

Features:

* Sidebar navigation styling
* Responsive layout
* Professional color scheme
* Typography and spacing
* Card-based content sections
* Hover effects and animations

The CSS ensures a consistent and professional appearance across all website pages.

---

## Website Structure

Website files are stored in:

```bash
cd /var/www/html
```

Main files:

```text
index.html
assignment1.html
assignment2.html
about.html
style.css
```

---

## Outcome

The website successfully provides a professional interface for accessing project documentation, assignment deliverables, and real-time monitoring resources. The implementation demonstrates practical web development skills using HTML, CSS, Linux hosting, and Nginx web server technologies.
